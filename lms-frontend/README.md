# LMS Frontend - Interface Utilisateur

Ce dossier contient l'interface utilisateur de la plateforme LMS, développée avec React et intégrée au monorepo global.

## Fonctionnalités Principales
* Interface Administrateur : Pilotage des structures académiques et des comptes.
* Interface Professeur : Gestion des cours, création de quiz et suivi des notes.
* Interface Étudiant : Accès aux ressources, évaluations et dashboard de progression.

## Prérequis
* Node.js (version 18 ou supérieure)
* npm

## Installation et Développement
1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Lancer l'environnement de développement :
   ```bash
   npm run dev
   ```

L'application est configurée pour communiquer avec l'API Gateway sur le port 8080 par défaut.
