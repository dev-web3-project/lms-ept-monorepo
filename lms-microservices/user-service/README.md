# User Service

Ce microservice est le cœur de la gestion des identités et des profils de la plateforme.

## Fonctionnalités
- Gestion des utilisateurs (Étudiants, Professeurs, Admins)
- Profils détaillés et rôles
- Système de gamification (XP, Badges, Streaks)
- **Gestion du Mentorat** (Demandes, Noms complets, Statuts)

**Port par défaut :** `8081`
