# University Service

Ce microservice gère l'architecture académique de l'établissement (Établissements, Facultés, Départements, Génies, Classes).

## Fonctionnalités
- Gestion des structures administratives
- Organisation des cycles d'études (Structure EPT : Cycle > Département > Classe)
- Cartographie des filières et classes

**Port par défaut :** `8082`
